# Wazuh SIEM Deployment & Security Monitoring

## Overview
Deployed and configured a Wazuh SIEM environment for centralized log collection, monitoring, and threat detection.

## Tasks Completed
- Installed Wazuh Manager, Indexer and Dashboard
- Configured Wazuh Agent
- Connected agent to manager
- Verified agent registration
- Monitored security events through the Wazuh Dashboard
- Troubleshot agent connectivity issues and other issues with indexer and manager
- Analyzed collected logs and alerts
